AI assist settings (optional)

Everything above runs on plain, built-in logic — no AI, no server, no cost. If you want an AI-generated answer for questions that don't match any built-in calculator or fact, you can optionally connect a free Google Gemini API key (no card required) from ai.google.dev. The key is stored only in this browser and sent directly from this device to Google — never to any server of ours. Anyone with access to this browser could read it back out, so only use a key you're comfortable having on this device.

Jit Pro

Jit Pro adds extras like PDF export on top of the free core. It's currently free for everyone while this launches — no license needed yet. Once it's for sale, it'll be a one-time unlock via a license key from a Gumroad purchase, verified directly against Gumroad's license API from this browser (still no server of ours involved, and your license key stored only on this device).
